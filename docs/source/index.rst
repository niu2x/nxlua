Welcome to NXLUA's documentation!
===================================

What's it 
---------------------------------------
- It's lua-5.1 with extra builtin libraries.
   - Image operation
   - socket
   - encrypt/decrypt
   - digest(eg: md5/sha1)
   - bit operation

- Is's fully build statically . No run-time dependencies.
